Actions

Main screen

To add a new list press the button .
To open the list just click it.
Menu "Save": Write all data to a json-file
Menu "Load": Export lists from a json-file. Lists from the file will be appended to the existed lists

List screen

To add a new item press the button .
To check/uncheck the item just click it.
To open the item use long click.
The list could be saved in a file in XML or JSON format
