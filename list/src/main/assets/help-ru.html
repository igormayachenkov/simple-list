Главное окно

Для добавления нового списка нажмите кнопку .
Чтобы открыть список просто нажмите на него.
Меню "Сохранить": Сохранить все данные json-файл
Меню "Импорт": Загрузить списки из json-файла. Данные из файла будут добавлены к существующим.

Окно списка

Для добавления нового элемента нажмите кнопку .
Чтобы отметить/снять отметку с элемента нажмите на него.
Чтобы открыть элемент для редактирования используйте длинное нажатие.
Список можно сохранить в файл в формате XML или JSON
